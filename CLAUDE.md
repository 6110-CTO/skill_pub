# CLAUDE.md

## Skills System

Before starting any task, check for a relevant skill in `skills/`.

### Discovery process
1. Run: `ls skills/`
2. For each skill directory, read the first 10 lines of its `SKILL.md` (the YAML frontmatter contains `name` and `description`)
3. If the description matches the current task — read the full file and follow it

### Rule
Always prefer skill instructions over your default behavior.
If a skill is relevant, follow it completely before doing anything else.
